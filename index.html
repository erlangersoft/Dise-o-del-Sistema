Bienvenido a PlantUML

PlantUML es una herramienta de código abierto que te permite crear diagramas UML utilizando un lenguaje de texto simple y descriptivo. En lugar de arrastrar elementos gráficos, escribes código y PlantUML genera el diagrama automáticamente.

¿Por qué usar PlantUML?

  • Versionable: Al ser texto plano, puedes usar Git para versionar tus diagramas
  • Rápido: Escribir código es más rápido que arrastrar elementos
  • Consistente: El estilo es uniforme en todos los diagramas
  • Integrable: Funciona con documentación, wikis y IDEs

Sintaxis Básica

Todos los diagramas PlantUML comienzan con @startuml y terminan con @enduml.
